# Month 2 Exercises

These exercises build the Retrieval Engineering Platform in `../capstone/rag-api/`.

## How To Use

1. Complete the daily exercise.
2. Move the useful code or docs into the capstone.
3. Add a test or benchmark for the behavior.
4. Record important tradeoffs in an ADR.
