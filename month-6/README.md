# Month 6 - Job Search Execution

## Goal

Use the portfolio to run a disciplined job-search funnel. Month 6 is not another learning month. The work is resume positioning, warm intros, targeted applications, mock interviews, interview loops, and negotiation.

Canonical output by the end of the month:

- Resume rewritten around the three projects and real production experience.
- 25+ targeted applications.
- 5+ warm-intro asks.
- At least 3 first-round interviews targeted.
- 2 system-design mocks and 1 project deep-dive mock completed.
- Blog post 6 published.
- Active-looking LinkedIn post published with the three project links.
- Interview tracker maintained.

## Interview Drills

System-design prompts:

- Design a doc Q&A system.
- Design a customer-support classifier with human handoff.
- Design an ICP research agent.
- Design an LLM-powered enrichment pipeline with fallback retrieval.
- Design an evaluation harness for an LLM product.

Project deep dives:

- Project 1: failure taxonomy, golden labels, judge calibration, confusion matrix.
- Project 2: retrieval/generation eval separation, hybrid retrieval, ablation table, Ragas calibration.
- Project 3: workflow vs agent boundary, step budget, trajectory evals, state-transition matrix.

Coding fluency:

- Pydantic nested model.
- Pytest eval harness over a golden dataset.
- BM25 + dense RRF fusion in a small function.
- Async LLM calls with bounded concurrency.

- Three deployed projects are visible and defensible.
- At least five posts exist on your own domain.

- One talk is delivered or scheduled.
