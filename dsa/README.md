# dsa/ — LeetCode arc (Weeks 5–20)

Per the [ROADMAP.md DSA standing rule](../ROADMAP.md): from Month 2, two weekday kata slots become **2 LeetCode problems/week**, committed here. Target **60–80 total by Month 6** so live-coding screens (possible from Month 5 when outbound opens) never catch you cold.

## Rules

1. **No AI.** These are interview reps. Solve on leetcode.com, then commit your solution here.
2. **~20 min cap.** Stuck past 25 min → read the editorial approach (not the code), close it, re-solve from a blank file. Log it as a re-solve, not a fail.
3. **Commit the same night.** Green squares are a Mercor screen filter — the commit is part of the rep.
4. **One-line pattern comment** in every file docstring — naming the pattern is what makes it transfer.
5. File naming: `w05_two_sum.py` (week number + slug). Copy `_template.py`.
